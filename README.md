# Usage 

## Windows

Start NNServer:

1. install python 3.x
2. pip install numpy scipy tensorflow keras
3. pip install flask flask_cors
4. cd NNServer/src; python app.py

Start front_end:
1. install node.js
2. cd front_end; npm install; npm start
